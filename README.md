# 🧮 Matrix Pro Desktop - Engineering Edition 
 
O **Matrix Pro Desktop** é uma ferramenta avançada de processamento matricial voltada para engenharia e ciência de dados. Desenvolvida para oferecer máxima performance, a aplicação encapsula o poder do motor **NumPy** em uma interface gráfica moderna, segura e intuitiva. 
 
--- 
 
## 🚀 Diferenciais de Engenharia 
 
- **Motor de Cálculo NumPy:** Utilização de algoritmos otimizados para álgebra linear (SVD, Determinantes, Inversas e Decomposições). 
- **Segurança de Dados (No-Eval):** Processamento de entradas via parser JSON, eliminando vulnerabilidades de execução de código e garantindo integridade. 
- **Arquitetura Desacoplada:** Separação clara entre a lógica matemática (`MatrixEngine`) e a interface visual (`MatrixApp`), facilitando a manutenção. 
- **UX Industrial:** Interface em Dark Mode com grid responsivo e console de saída em fonte monoespaçada para alinhamento perfeito de colunas matriciais. 
- **Tratamento de Exceções:** Sistema resiliente a erros comuns (matrizes singulares, dimensões incompatíveis ou entradas inválidas). 
 
## 🛠️ Tecnologias Utilizadas 
 
- **Python 3.x** 
- **NumPy:** Computação científica e processamento de arrays multidimensionais. 
- **CustomTkinter:** Framework para UI moderna e profissional. 
- **JSON Parser:** Manipulação segura de strings para estruturas de dados. 
 
## 🎮 Funcionalidades Operacionais 
 
- **Operações Básicas:** Soma, Subtração e Multiplicação escalar/matricial. 
- **Análise Avançada:** Cálculo de Posto (Rank), Traço, Determinante e Matriz Inversa. 
- **Transformações:** Transposição de matrizes em tempo real. 
- **Sistema de Log:** Console interno para exibição detalhada de resultados matemáticos.
